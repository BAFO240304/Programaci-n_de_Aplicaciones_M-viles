Manual de Instalación

Para instalar Flutter en tu máquina, sigue estos pasos:

  1. Descarga el SDK de Flutter desde la página oficial.
  2. Extrae el archivo descargado y añade la ruta a tu variable de entorno PATH.
  3. Ejecuta el comando flutter doctor en tu terminal para verificar que todo esté correctamente configurado.
  4. Configura un editor de texto compatible, como Visual Studio Code o Android Studio.
